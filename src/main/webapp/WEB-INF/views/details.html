领养宠物须知

来源：宠物领养网 作者：www.cwlyz.com 编辑：xiaogui 编辑时间: 2018-6-1 19:48

  1. 领养宠物回家首先考虑宠物本身，它至少要两个月大，而且领养那一天狗的状态非常好，愿意和人接近。
  2. 从自己角度考虑，因小宠物刚离开妈妈和兄弟姐妹，所以开始的几天需要你加倍的呵护和照顾。如果你是一个上班族，最好选择一个周末去接狗狗，这样就有周六和周日两天来照顾它。小狗狗对第一个照顾它的人感情最深，这样的安排可以让狗狗认识你，加深它和你的感情。
  3. 领养宠物的时间最好是早上。早上去接受宠物，在宠物没吃早饭时把它抱走。因为他们吃饱了，路上再一折腾，非常容易呕吐。
  4. 从对新环境的适应性来讲，如果能一下领养两只小狗是最好的，它们可以互相鼓励，携手相伴长大，而且也不会养成对主人过分依赖的性格。同时养两只，小狗的成活率大大高于只养一只小狗 或者你家里 已经有一只了 也可以考虑再领养一只。
